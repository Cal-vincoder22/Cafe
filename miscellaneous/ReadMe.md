# Café Mocha Website
This is a static HTML website for Café Mocha, a local café and bakery in Johannesburg, South Africa.

## Website Structure
- Homepage (index.html): Welcome section, specialties overview, and call to action
- About Us (about.html): Company story, mission, values, and team information
- Menu (menu.html): Detailed menu with coffee drinks, pastries, and retail coffee beans
- Contact (contact.html): Location details, contact information, and interactive map
- Enquiries (enquiry.html): Contact form for customer inquiries and questions

## Features
Responsive navigation across all pages
Visual elements showcasing coffee products and café atmosphere
Contact form with multiple enquiry types
Google Maps integration for location finding
Consistent branding and footer across all pages

## Technical Details
Built with HTML5
Semantic markup for better accessibility
Responsive design considerations
Relative paths for asset management

## Overview
This CSS file provides the complete styling for Café Mocha's website, featuring a warm, coffee-themed design with responsive layouts, animated social media icons, and an elegant menu card system.

## File Structure
The CSS is organized into the following sections:
- Base Styles - Global styles and resets
- Header & Navigation - Top navigation and branding
- Menu Card System - Product display cards with hover effects
- Social Media Icons - Animated social media links with tooltips
- Footer - Bottom section styling
- Responsive Design - Media queries for different screen sizes

# customization
- Size: Adjust the width and height properties to change icon size
- Colors: Modify the background and color properties for different themes
- Animation: Adjust transform values and transition timing for different effects
- Tooltips: Change the data-platform attribute values to customize labels
